# 🛠 기술 스택
- Java 21
- Spring Boot 3.4.4
- Spring Security
- JPA
- H2 Database

# 💡 기능
- 사용자 회원가입
  - 일반 사용자 권한을 받습니다.
- 관리자 회원가입
  - 관리자 권한을 받습니다.
- 로그인
- 사용자 전체 조회
  - 관리자는 모든 사용자의 정보를 조회할 수 있습니다.
    - 이메일, 별명, 권한
- 공지사항 작성
  - 관리자는 공지사항을 작성할 수 있습니다.
- 공지사항 조회
